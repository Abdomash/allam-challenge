## Usage

### Backend Python UI:

Install dependencies:
`pip install -r requirements.txt`

Running the backend server:
`python website.py`

If you want a CLI (Command-Line Interface), you can run the program like this:
`python main.py`

### Frontend React UI:

Install dependencies:
`cd allam-website && npm i`

Running the frontend:
`npm run build`
